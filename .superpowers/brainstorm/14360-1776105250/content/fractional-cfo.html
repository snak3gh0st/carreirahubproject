How the CFO Brain Works
📊
1. Rule-Based Analysis
No AI needed. Pure math + business rules:

• Collection rate drops > 3% → flag
• Invoice > 30 days overdue → escalate
• Top N clients > 40% revenue → concentration warning
• Payment pattern degrading → early warning
• 30-day cash < projected expenses → alert
🔍
2. Pattern Detection
Compare current vs historical:

• Customer X paying slower each month?
• Revenue seasonality (same month last year)?
• Auto-charge failure rate increasing?
• Collection calls yielding fewer commitments?
• Which services growing vs shrinking?
💡
3. Actionable Output
Every insight links to an action:

• "Escalate X" → links to invoice
• "Watch Y" → links to customer
• "Cash tight" → lists which invoices to push
• "Concentration high" → suggests strategy
• Monthly briefing → exportable PDF
No OpenAI / LLM dependency. This is deterministic logic — thresholds + comparisons + templates. Fast, predictable, no API costs. The "intelligence" comes from knowing what a CFO would look at, not from AI generation.
Where CFO Intelligence Appears
CFO Briefing Banner
Top of page. 2-3 sentence narrative summary of financial health. Plain English. Updated with each page load.
Action Items
Below briefing. Prioritized list: URGENT (needs action today), WATCH (degrading trend), INSIGHT (strategic observation). Max 5 items.
KPI Context Labels
Each KPI card gets a one-line context: "▼ 5% — needs attention" instead of just "▼ 5%". Cards get colored borders when metrics cross thresholds.
PDF Monthly Report
Export includes the CFO briefing as the opening paragraph. CEO can forward to board or investors without additional context.
