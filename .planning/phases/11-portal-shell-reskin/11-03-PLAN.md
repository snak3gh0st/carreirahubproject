Target design pattern (from research):
```tsx
// Verde full-bleed bg -> Creme card -> Tangerina CTA
<div className="min-h-screen flex items-center justify-center bg-brand-verde p-4">
  <Logo mono className="text-brand-creme" />
  <h1 className="font-display text-3xl font-bold text-brand-creme">
  <div className="bg-brand-creme rounded-2xl shadow-sm p-6">
    <button className="bg-brand-tangerina text-white">
```

WCAG constraints:
- text-brand-tangerina on Creme/white = FAIL (~2.9:1)
- text-brand-verde on Creme = PASS (~7.5:1)
- Use text-brand-verde for links on Creme cards
- Use bg-brand-tangerina text-white for CTA buttons
