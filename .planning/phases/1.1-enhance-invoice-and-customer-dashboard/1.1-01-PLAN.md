  3. Calculate aggregated stats:

     - Paid invoices count and amount (status = PAID)

     - Overdue invoices count and amount (status != PAID AND dueDate < today)
     - Outstanding balance (total invoiced - total paid)

  **Page Layout:**
  - Header: Customer name, email, phone

  - Financial Summary Cards (4-column grid on desktop):

    3. Pending: amount + count (yellow)
    4. Overdue: amount + count (red)
  - Back button to /dashboard/customers

  **Implementation notes:**
  - Use Prisma `findUnique({ where: { id }, include: { invoices: true } })`

  - Handle customer not found: show 404 message
