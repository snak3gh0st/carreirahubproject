# Plan 12-02: Recharts Brand Palette + Tooltip Typography

## Objective

All Recharts chart components across the dashboard render using the Carreira USA brand palette (Verde, Tangerina, Cafe com Leite, Caramelo). Zero hardcoded legacy hex values remain in chart files. Chart tooltips use Neue Montreal typography with Creme background and Verde border.

## Success Criteria

- [ ] SC1: All Recharts chart components use BRAND_COLORS constants — zero hardcoded hex in chart files
- [ ] SC2: Chart tooltips use Neue Montreal font-family, Creme (`#F5F0E8`) background, Verde (`#1B4332`) border

## Brand Palette to Use

```ts
// From lib/constants/brand.ts
BRAND_COLORS = {
  VERDE:           "#1B4332",  // primary
  TANGERINA:       "#E8681A",  // accent
  CREME:           "#F5F0E8",  // background
  CAFE_COM_LEITE:  "#C4A882",  // neutral warm
  CARAMELO:        "#8B6914",  // warm accent
}
```

Chart series palette order: Verde → Tangerina → Cafe com Leite → Caramelo → Verde (lighter variants)
